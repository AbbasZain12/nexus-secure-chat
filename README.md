# Nexus Secure
A real-time, end-to-end encrypted chat application built with the MERN stack (PostgreSQL).

## Features
* 🔒 End-to-End Encryption (CryptoJS)
* 🤖 AI Smart Scam Detection
* 🛡️ Dynamic Trust Scoring & Verified Badges
* 💻 Live Code Pair-Programming (Monaco Editor)
* 🎨 Real-Time Collaborative Whiteboard
* 🎥 WebRTC Video & Audio Calling

## How to run locally
1. Clone the repo
2. Run `npm install` in both `/client` and `/server`
3. Create a `.env` file in the server with your PostgreSQL credentials
4. Run `npm run dev` in both folders.
